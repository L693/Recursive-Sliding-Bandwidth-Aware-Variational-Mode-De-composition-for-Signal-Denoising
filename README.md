# README вЂ” Dataset Structure, Formats, and Interpretation Guide

## 1. Dataset Overview

This code repository implements and evaluates the **Recursive Sliding-window Band-Adaptive Variational Mode Decomposition (RSBAVMD)** algorithm for vibration signal denoising. It supports two scenarios:

| Scenario | Script    | Signal Type | Purpose                                                    |
|----------|-----------|-------------|------------------------------------------------------------|
| A        | `demo1.m` | Synthetic   | Full denoising benchmark: RSBAVMD vs. VMD, EMD, filtering  |
| B        | `demo2.m` | Real only   | RSBAVMD decomposition and visualization of measured data   |

Both scripts are **self-contained** вЂ” all helper functions are defined at the bottom of each file. No external `.m` file dependencies are required beyond standard MATLAB toolboxes.

### 2.1 `data.mat` Format

The data file is a MATLAB binary `.mat` file (v7 format) containing a single variable:

```
Name        Size            Bytes   Class 
Sig      10000x1            80000   double
```
- **Variable name**: `Sig`
- **Type**: double-precision floating-point array
