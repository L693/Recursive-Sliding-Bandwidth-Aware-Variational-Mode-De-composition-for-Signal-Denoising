# README — Dataset Structure, Formats, and Interpretation Guide

## 1. Dataset Overview

This code repository implements and evaluates the **Recursive Sliding-window Band-Adaptive Variational Mode Decomposition (RSBAVMD)** algorithm for vibration signal denoising. It supports two scenarios:

| Scenario | Script    | Signal Type | Purpose                                                    |
|----------|-----------|-------------|------------------------------------------------------------|
| A        | `demo1.m` | Synthetic + Real | Full denoising benchmark: RSBAVMD vs. VMD, EMD, bandpass filtering |
| B        | `demo2.m` | Real only  | RSBAVMD decomposition and visualization of measured data   |

Both scripts are **self-contained** — all helper functions are defined at the bottom of each file. No external `.m` file dependencies are required beyond standard MATLAB toolboxes.

### 2.1 `data.mat` Format

The data file is a MATLAB binary `.mat` file (v7 format) containing a single variable:

```
Name        Size            Bytes   Class     Attributes
Sig      10000x1            80000   double
```
- **Variable name**: `Sig`
- **Type**: double-precision floating-point array

## 3. Script Execution Flow

### 3.1 `demo1.m` — Execution Outline

```
demo1.m
│
├── [Section 1] Simulated Signal Generation
│   ├── Construct 3-component composite signal (damped mode + 2 intermodulation tones)
│   ├── Add white Gaussian noise at SNR = 2 dB
│
├── [Section 2] RSBAVMD Decomposition
│   ├── Compute FFT and prepare SigNewAdd (50-sample sliding step)
│   ├── Run RSBAVMD
│   └── Display: Figure 2 — RSBAVMD modes + spectra
│
├── [Section 4] VMD Decomposition (K=5, alpha=2000)
│   ├── Run standard VMD
│   └── If VMD fails, fallback to x_noisy
│
├── [Section 5] EMD Decomposition
│   ├── Run EMD
│   └── If EMD fails, fallback to x_noisy
│
├── [Section 6] Bandpass Filtering
│   ├── Use frequency bands detected by RSBAVMD spectrum analysis
│   ├── Apply Butterworth 4th-order bandpass per band, sum results
│   └── If filtering fails, fallback to x_noisy
│
├── [Section 7] Metrics Calculation
│   ├── Per-component metrics: MPE, SNR, correlation
│   ├── Per-method metrics: same three metrics for RSBAVMD/VMD/EMD/Bandpass
│   ├── Baseline metrics: MPE/S/SNR/correlation for noisy and clean signals
│   └── Display: Figure 3 — Clean vs. Noisy vs. Denoised comparison
│
└── [Functions] All helper functions defined inline
```

## 6. Citation and References

If you use this code in academic work, please cite the corresponding paper on RSBAVMD signal denoising. The algorithm is based on:

- **Variational Mode Decomposition (VMD)**: Dragomiretskiy, K., & Zosso, D. (2014). "Variational Mode Decomposition." *IEEE Transactions on Signal Processing*, 62(3), 531–544.
- **Permutation Entropy**: Bandt, C., & Pompe, B. (2002). "Permutation Entropy: A Natural Complexity Measure for Time Series." *Physical Review Letters*, 88(17), 174102.
- **Recursive FFT**: Dan, D.; Zeng, G.; Pan, R.; Yin, P. Block-Wise Recursive Sliding Variational Mode Decomposition Method and Its Application on Online Separating of Bridge Vehicle-Induced Strain Monitoring Signals. Mech. Syst. Signal Process. 2023, 198, 110389.
